🖨️ Raspberry Pi Wireless Print Server Setup

This guide helps you turn a Raspberry Pi and USB printer (like HP LaserJet P1106) into a wireless print server for Windows and Android.

📦 Requirements

  • Raspberry Pi 4 with Raspberry Pi OS
  • USB Printer (e.g., HP LaserJet P1106)
  • Wi-Fi network

⚙️ Step-by-Step Setup

  1. sudo apt update && sudo apt upgrade -y
  2. sudo apt install cups -y
  3. sudo usermod -aG lpadmin rk
  4. Edit the CUPS config file: 
    sudo nano /etc/cups/cupsd.conf
  5. Make the following changes:
    • Add Allow @local under these sections:
      • <Location />
      • <Location /admin>
      • <Location /admin/conf>
      • <Location /admin/log>
  6. Restart the CUPS service: 
    sudo systemctl restart cups
  7. sudo apt install hplip hplip-gui -y
  8. sudo hp-setup -i → Select USB and follow the prompts
  9. Open http://<Pi-IP>:631 in a browser
  10. Add printer and select correct driver (HP LaserJet Professional P1106)
  11. Set that printer as the default
  12. Delete duplicate or non-working printers (if any)

🖥️ Wireless Printing from Windows

  • Go to Control Panel → Devices and Printers → Add Printer
  • Click “The printer that I want isn't listed”
  • Choose “Select a shared printer by name”
  • Enter: http://<Pi-IP>:631/printers/HP-LaserJet-Professional-P1106
  • Select appropriate driver or use Windows Update to find HP P1106

📱 Wireless Printing from Android

  • Install Mopria Print Service from Play Store
  • Go to Settings → Printing → Enable Mopria
  • Tap “Add printer” manually
  • Enter IPP URI: ipp://<Pi-IP>:631/printers/HP-LaserJet-Professional-P1106
  • Save and print
